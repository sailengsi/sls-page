简介

首先，这只是一个分页算法类，因为每个项目的分页样式都不同，所以不会给你生成完整的分页dom，但这个类会提供给你所有需要的分页数据。

再使用之前，您必须要知道以下几点关于分页的概念

  • page_total 您的数据总数
  • page_size 每页显示的数量
  • page_count 您的数据能分的最大页数
  • page_cur 当前显示的页数
  • page_show 显示的页码个数
