安装

安装

GitHub: src中是源码，使用es6编写。dist/sls-page.js是最终打包后可使用的代码。

git clone git@github.com:sailengsi/sls-page.git

浏览器环境

<script src="/path/to/sls-page.js"></script>

NPM

npm install sls-page
import SlsPage from 'sls-page';
